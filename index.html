Command-Line Port Instructions

Essential commands for checking port status and security on Linux/macOS and Windows.

Check Listening Ports

View what services are running and on which ports.

# Linux/macOS
ss -tuln
# Windows (find PID)
netstat -ano | findstr LISTENING

Security Scanning (Nmap)

Scan for open ports on a target system.

# Scan common ports
nmap -F 192.168.1.1
# Scan a specific range
nmap -p 1-1024 target.com

Firewall Rule (UFW)

Open or close ports using `ufw` on Ubuntu/Debian.

# Open Port 80 for HTTP
sudo ufw allow 80/tcp
# Deny Telnet
sudo ufw deny 23
